# Student-registration-form
A simple desktop-based Student Registration application built using Java Swing.

## Features
- GUI-based form
- Input validation
- Modular code structure
- Object-Oriented design

## Technologies Used
- Java
- Swing (JFrame, JPanel, JButton, JTextField)

## How to Run
1. Compile all files:
   javac *.java
2. Run:
   java Main

## Learning Outcomes
- Java Swing GUI
- Event handling
- MVC-style separation
- Clean project structure
